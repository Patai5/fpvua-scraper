# FPVUA Scraper

Scrapes product listings from [fpvua.com](https://fpvua.com) — a Ukrainian FPV drone parts store. Point it at any category and it will paginate through every page, pulling structured product data from each listing.

Built with [Crawlee](https://crawlee.dev/) and [Cheerio](https://cheerio.js.org/), runs on the [Apify platform](https://apify.com).

## Usage

```bash
apify run    # run locally
apify push   # deploy to Apify
```
